Dentro del HTML crear un script que contenga un arreglo llamando array1 con los siguientes valores: 3, 1, 7, 4, 9, 0 Crear un segundo arreglo llamado array2 con los siguientes valores: 2, 6, 5, 8 Y por último crear un JSON en forma de String con las siguientes propiedades y valores: nombre = “Carlos” apellido= ”Vergara” A continuación mostraremos las funcionalidades de los botones: Al hacer clic en el botón “Mostrar Arreglos” se recorrerá los arreglos array1 y array2 para mostrar sus elementos en el HTML. Al hacer clic el botón “Combinar Arreglos” se combinarán los elementos del array2 con los del array1 para luego mostrar el arreglo combinado y sus elementos en el HTML. Al hacer clic en el botón “Filtrar elementos” se realizará un filtro en el arreglo combinado que consiste en filtrar los elementos que sean mayores que 4, y se recorrerá el nuevo arreglo filtrado para mostrar los elementos en el HTML. Al hacer clic en el botón “Convertir JSON” se convertirá en un objeto JSON el String JSON que habíamos creado inicialmente para luego recorrerlo y mostrar sus propiedades y valores en el HTML.
